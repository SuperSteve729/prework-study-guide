HTML

  • Head element contains all the metadata of the site, this is not visible to the user
  • The meta element contains most of the important data used for the browser
  • Body element contains the structure of the website, this is visible to the user
  • You need to add elements like p or ul to have formatting on the website so it looks clean to the user and developers

GIT

  • Most GIT commands start with git
  • git status: check what branch you currently are on
  • git checkout -b branch-name: this creates a new branch with your selected name, then swaps to said branch
  • git add -A: this adds all new content added to the currently used branch to be staged for a commit
  • git commit -m "your message here": this commits to the branch, acting as a version of the branch, with an included short message
  • git pull origin branch-name: this will pull any new content from the selected branch to your local files
